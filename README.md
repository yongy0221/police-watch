# Installation Instructions

1. Install [Node.js](https://nodejs.org/en/download/).
2. Install MySQL.
   ```bash
   npm install mysql
   ```
3. Set up localhost password for MySQL.
   ```bash
   sudo apt-get install mysql-server
   ```

4. Install Bootstrap
   Instructions will be posted later.

5. Run on localhost.
   ```bash
   npm start
   ```
